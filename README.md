# Turn-Based Witcher-Inspired Combat Prototype

## Overview
This is an early prototype of a turn-based combat system inspired by The Witcher.
The project focuses on turn resolution, player/enemy actions, and basic combat flow.

## Current State
- Probabilistic combat resolution
- Turn-based player vs enemy battles
- Basic enemy AI
- XP and leveling system

This version is a prototype and not yet balanced.

## How to Run
```bash
python game.py
